Requirements:
1) Python 2.x - http://www.python.org/download/
2) Pygame - http://www.pygame.org/download.shtml

Changes made:
1) Tweaked visuals
	- Each snake unit is now 10x10 pixels
	- Each snake unit is now a unfilled green rectangle
	- Food is now a solid red 10x10 rectangle
	- Screen is 600x600 pixels

2) Added ability to wrap across the boundaries
3) Score is now in multiples of 10
4) Local multiplayer capability (shared keyboard)
5) Pause function (ESC to pause)
